# CodeAlpha Internship Tasks 🚀

## Task 1: Language Translator 🌐
A simple web-based translator built with HTML, CSS & JavaScript.
**Live Demo:** https://pramelanarvakar-pixel.github.io/CodeAlpha-Language-Translator/

**Features:**
- English ↔ Hindi translation
- English ↔ Marathi translation  
- Mobile-friendly responsive design
- No API key required - works offline with dictionary

## Task 2: ChatBot 🤖
An interactive chatbot for CodeAlpha queries.
**Live Demo:** https://pramelanarvakar-pixel.github.io/CodeAlpha-Language-Translator/chatbot.html

**Features:**
- Responds to greetings, help, task info
- Shows current time
- Clean chat UI with user/bot bubbles
- Linked with Translator page

## Tech Stack
- HTML5, CSS3, JavaScript
- Hosted on GitHub Pages
- 100% Frontend - No backend needed
